Privacy Policy

Last updated: April 18, 2026

PWMarketing is a personal automation tool operated by Eric Spiegelman for private, single-user use. This policy explains what information the tool handles and how.

Who operates this tool

PWMarketing is a personal project by Eric Spiegelman. It is not a commercial product and has no other users.

Information accessed

When connected to a TikTok account, PWMarketing may access:

  • Basic profile information (username, display name, avatar) via the user.info.basic scope
  • The ability to upload video content to the account's inbox or publish directly, via the video.upload and/or video.publish scopes

The tool only accesses the account of the operator (Eric Spiegelman) and does not collect information about any other TikTok users.

How information is used

Access tokens and profile information are used solely to upload and post videos to the operator's own TikTok account. No data is sold, shared, or transferred to any third party.

How information is stored

Access tokens are stored locally on the operator's own machine or in a private GitHub repository's encrypted secrets. No user data is stored on any public server or shared database.

Data retention and deletion

Access tokens are retained only as long as needed for the tool to function and can be revoked at any time by disconnecting the app in TikTok settings. Any stored profile information is deleted when the tool is no longer in use.

Third-party services

PWMarketing uses the TikTok Content Posting API. Use of that API is governed by TikTok's own privacy policy.
